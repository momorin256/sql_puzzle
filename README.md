# Setup

## PostgreSQL

```sh
$ cd postgres
$ docker-compose up -d
$ docker-comopose exec postgres psql -U user
```

## MySQL

```sh
$ cd mysql
$ docker-compose up -d
$ docker-compose exec mysql mysql -prootpass -uroot
```
